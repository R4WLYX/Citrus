# Citrus
Citrus is an OpenGL based game engine using Entity Component System. This game engine has been in develepmont since May 25 2023.

## Usage
Run the command.<br>
```bash
git clone https://github.com/r4wlyx/Citrus
```

To compile run.<br>
```bash
citrus <file> -o <file>
```
